# Hair PRP Sessions - Ajman

A professional website for a Platelet-Rich Plasma (PRP) hair restoration business in Ajman, UAE.

## Features

- **Modern Design**: Clean, responsive design optimized for all devices
- **Multiple Pages**: Home, Services, Locations, About, and Contact pages
- **SEO Optimized**: Meta tags and descriptions for better search engine visibility
- **Fast Loading**: Built with Astro for optimal performance
- **Contact Form**: Easy-to-use contact form for customer inquiries

## Pages

1. **Homepage** - Introduction to PRP hair restoration services
2. **Services** - Detailed information about treatment packages and additional services
3. **Locations** - Clinic locations with addresses and hours
4. **About** - Company story, team members, and approach
5. **Contact** - Contact form and clinic information

## Technologies Used

- [Astro](https://astro.build/) - Modern static site builder
- HTML5 & CSS3 - For structure and styling
- JavaScript - For interactive elements

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/hair-prp-ajman.git
   ```

2. Navigate to the project directory:
   ```bash
   cd hair-prp-ajman
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```

## Deployment
